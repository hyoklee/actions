HDF5 Packet Table APIs

Introduction

The HDF5 Packet Table API (H5PT) provides an interface for storing and retrieving fixed-length packets of data in append-only datasets. Packet tables are optimized for streaming data collection where records are added sequentially.

See also
HDF5 Packet Table APIs (H5PT) Reference Manual

Packet Table Operations

Create packet tables with H5PTcreate or open existing ones with H5PTopen. Append packets using H5PTappend and read packets with H5PTread_packets. Get the number of packets with H5PTget_num_packets. Close with H5PTclose.
